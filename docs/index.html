Worum geht es in unserem Spiel?

Unser Spiel besteht aus zwei Teilen. Die zwei Figuren kämpfen im zweiten Teil gegeneinander. Dafür müssen sie im ersten Teil zuerst durch zwei unterschiedliche Labyrinthe gehen. Dabei müssen sie Elemente aufsammeln, die ihnen beim Kampf helfen können.


Wo ist unser Spiel verfügbar?

Unser Spiel können Sie auf userer Website oder mit unserem Repository spielen.
